# new

new 关键字：  
对于任意类型，都使用new进行动态内存分配。例如：  
`
	int* num = new int(5);
	ListNode* l = new ListNode(sum)
`
其中，这个ListNode表示链表中的一个结点。  
可见，使用new修饰后 表示的是 从内存中去一段空间，放入括号内的值，然后用定义的指针指向这段内存。  
所以，在链表中，使用new后实实在在地开出了一段内存当作了链表的结点，但没有变量名对应了这个结点，而是有相应类型的指针temp、tail等指向了它。  

# nullptr  

nullptr表示空指针，能够隐性转化任何指针类型，避免类型不匹配的问题。
